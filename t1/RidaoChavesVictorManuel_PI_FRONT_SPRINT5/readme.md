# SPRINT 5: MODELADO PLANTUML

## Descripción
Este repositorio contiene el FrontEnd de la aplicación **Calculadora de Calorías**, que permite:
- Añadir alimentos individualmente y calcular calorías de la comida.
- Guardar recetas propias y consultar recetas de otros usuarios.
- Funcionalidades de red social: favoritos, compartir recetas.

---

## Tecnologías

- PlantUML para diagramas de modelado (Sprint 5)

---
